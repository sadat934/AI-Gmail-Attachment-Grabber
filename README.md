# GrabMail Landing Page

This is the landing page for GrabMail - an AI-powered Gmail attachment downloader Chrome extension.

## 🚀 Deployment Options

### Option 1: GitHub Pages (Free & Easy)
1. Create a new repository on GitHub
2. Push these files to the repository
3. Go to repository Settings → Pages
4. Select "main" branch and "/" (root) folder
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Netlify (Free with custom domain)
1. Go to [netlify.com](https://www.netlify.com/)
2. Sign up/Login
3. Drag and drop this folder into Netlify
4. Your site will be live instantly with a netlify.app URL
5. Optional: Add custom domain in Netlify settings

### Option 3: Vercel (Free with custom domain)
1. Go to [vercel.com](https://vercel.com/)
2. Sign up/Login
3. Import your GitHub repository or upload folder
4. Your site will be live instantly with a vercel.app URL
5. Optional: Add custom domain in Vercel settings

### Option 4: Cloudflare Pages (Free with custom domain)
1. Go to [pages.cloudflare.com](https://pages.cloudflare.com/)
2. Sign up/Login
3. Connect your GitHub repository
4. Your site will be deployed automatically
5. Free SSL and CDN included
