What is GrabMail?

GrabMail is a powerful Chrome extension that revolutionizes how you download email attachments from Gmail. Whether you're a teacher collecting assignments, a professional managing invoices, or anyone organizing files from email, GrabMail saves you hours of manual work.

AI-Powered Search

Describe what you need in plain English and let AI find it. Supports Claude, Gemini, and OpenRouter.

  • Natural language understanding
  • Smart date parsing
  • Multiple AI providers
  • Intelligent filtering

Manual Search Mode

No AI? No problem. Use direct keyword, sender, file type, and date range filters.

  • Keyword filtering
  • Sender filtering
  • Multiple file types
  • Date range control

Smart Download Options

Choose between convenient ZIP archives or individual file downloads.

  • ZIP archive mode
  • Individual downloads
  • Automatic compression

Key Features

🤖 AI-Powered Search

Describe what you want in natural language and watch GrabMail find it instantly. Our AI understands context, handles complex queries, and learns from your descriptions.

  • Claude Sonnet 4: Best for complex queries
  • Gemini 2.5 Flash: Fast and free tier available
  • OpenRouter: Access to 100+ models

Example: "Download all PDF invoices from last month"

🔍 Manual Search Mode

No AI required! Use powerful filters for precise control: keywords, sender email, file types, and date ranges. Perfect for repetitive tasks and users who prefer not to use AI.

  • Keyword filter: Search subject and filename
  • Sender filter: Exact match or domain-based
  • File types: pdf, docx, xlsx, jpg, zip, apk, and more
  • Date range: 7 days to 1 year

📦 Smart Download Options

Choose the download format that works best for you. Get everything in one ZIP file or download each attachment individually to your Downloads folder.

ZIP Archive Mode

Each attachment downloads separately with original filenames preserved

Privacy First

Your Data Stays Secure

GrabMail is built with privacy as the foundation. We don't collect, store, or share your personal data. Here's what we do:

  • Only reads attachment metadata
  • Never reads email body content
  • No data stored on external servers
  • All processing happens locally
  • AI only sees: filename, sender, subject, file size

✓ Open source — review the code yourself
✓ GDPR, CCPA, and COPPA compliant
✓ Transparent permissions — only what we need
